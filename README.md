# Twitter Analytics Dashboard using Microsoft Power BI

## Overview

This project was completed as part of my Data Analytics internship using **Microsoft Power BI**. The objective was to analyze a Twitter dataset and build interactive dashboards that provide insights into tweet performance, user engagement, and audience interaction. The dashboards were developed by applying various business rules, DAX calculations, and data transformation techniques.

## Objectives

* Analyze Twitter engagement data using Power BI.
* Create interactive and visually appealing dashboards.
* Apply business requirements through filters and DAX measures.
* Identify trends and patterns in tweet performance.
* Improve data visualization and reporting skills.

## Dashboard Tasks

### Task 1 – Tweet Interaction Breakdown

Created a clustered bar chart to compare URL Clicks, User Profile Clicks, and Hashtag Clicks across tweet categories using predefined business conditions.

### Task 2 – Engagement Rate Comparison

Compared the average engagement rate of tweets with App Opens and without App Opens after applying multiple filtering conditions.

### Task 3 – Media Interaction Analysis

Developed a dual-axis chart to compare Media Views and Media Engagements by weekday for Quarter 4 data.

### Task 4 – Replies, Retweets, and Likes Comparison

Analyzed user engagement by comparing Replies, Retweets, and Likes for tweets posted between June and August 2020.

### Task 5 – Monthly Engagement Rate Trend

Created a line chart showing monthly engagement rate trends for tweets with media and tweets without media.

### Task 6 – Top 10 Tweets by Engagement

Identified the top-performing tweets based on the combined total of Likes and Retweets while following specified business rules.
